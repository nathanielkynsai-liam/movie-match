# Movie Match

## One-Sentence Pitch

Movie Match lets users discover movies tailored to their tastes so that they can quickly find something they'll enjoy watching.

## The User

Movie lovers who open the app when they want a movie recommendation, are unsure what to watch, or want to explore films similar to their favorites.

## Core Screens

### Home Screen

Shows featured and recommended movies.

### Search Screen

Allows users to search for movies by title or genre.

### Movie Details Screen

Displays information about a selected movie, including rating, genre, release year, and overview.

### Recommendations Screen

Shows AI-generated movie recommendations based on the user's preferences.

## The Data

### Movie

* Title
* Genre
* Rating
* Release Year
* Overview
* Poster URL

### User Preference

* Favorite Movies
* Preferred Genres
* Minimum Rating

## The AI Feature

The model takes a user's movie preferences and favorite films and returns personalized movie recommendations.
